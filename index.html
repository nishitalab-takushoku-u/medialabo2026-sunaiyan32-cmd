「Webコンテンツ作成」の成果

HTML・CSS・JavaScript を使って、情報を整理しながら見やすい Web ページを作る力を身につけることを目標に学習しています。 このページでは授業で学んだ内容と、演習で制作したページへのリンクをまとめています。

このテーマについて

このテーマでは、Web ページの構造を HTML で記述し、CSS で見た目を整え、必要に応じて JavaScript で動きを加える方法を学びます。 伝えたい内容に合わせて要素を選び、利用者が読みやすい形で情報を届けることの大切さを意識しながら制作しました。

第3回 JavaScript の基本

第3回では、外部 JavaScript ファイルを HTML から読み込む方法や、 コンソールで結果を確認する方法、オブジェクトや配列から必要なデータを取り出す書き方を学びました。 project.html 用の天気データでも、表示したい項目を選んで整理することを意識しました。

演習3-6: obj-practice.html

プロジェクト課題について

プロジェクト課題では「世界の天気」をテーマに選び、都市ごとの天気や気温、湿度、風の情報を確認できるページを作成しました。 表を使って情報を整理し、必要な項目をすぐ見つけられるページ構成を意識しています。

プロジェクト課題のページを見る

第1回 WWW と HTML

第1回では、Web ページが Web サーバからブラウザへ送られて表示される仕組みを学びました。 さらに、見出し・段落・リスト・リンク・画像などの基本要素を使い分けることで、内容に合った文書構造を作れることを理解しました。

演習1-4: HTML のさまざまな要素

第2回 CSS による Web ページの装飾

第2回では、HTML で作った文書に対して CSS を使い、文字の大きさや色、余白、枠線、配置を調整する方法を学びました。 同じ内容でも CSS を加えることで、読みやすさや印象が大きく変わることを実感しました。今回の課題では外部スタイルシートを利用し、 複数のページに共通したデザインを適用することで、サイト全体に統一感を持たせる練習もできました。
